# Changelog

All notable changes to LifeBoard are documented in this file.

## 2026-06-14

### Initial Release

- Added a local-first Home dashboard connected to real module state.
- Added weather search, current conditions, forecasts, and practical advice through Open-Meteo.
- Added locally persisted todos and countdowns.
- Added private in-browser text and data tools.
- Added locally persisted bookmarks with categories, search, and pinning.
- Added Settings controls for themes, local data review, backup, restore, privacy, and selective clearing.
- Added responsive navigation, light and dark themes, accessible states, and unknown-route recovery.
- Added release-readiness fixes for weather persistence, date rollover, theme initialization, documentation, metadata, and Vercel SPA routing.
